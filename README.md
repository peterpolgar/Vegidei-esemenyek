# Végidei események

A végidei események áttekintése 2025-től Jézus második eljöveteléig. Adventista próféciamagyarázat Ellen White írásai alapján.

Jelenleg csupán a jelentől (2025.12.23.) a kegyelemidő lezárulásáig terjedő események kerültek feldolgozásra, de reményeim szerint majd felkerül a kegyelemidő lezárulásától Jézus második eljöveteléig tartó események áttekintése is.

A jelenlegi egyetlen prezentációt nem mondanám teljesen késznek, a 107. diáig tudom azt mondani, hogy teljesen kész tartalmilag, utána csak nagyjából kész.
